# 武器详情页 (Weapon Detail Page)

**上下文**: 在 MVP 版本中，这是一个侧边栏面板。在未来版本中，可能独立为 `/weapon/[id]` 页面。

## 1. 功能概述
深入展示特定武器的详细统计数据，计算其理论性能上限与下限。

## 2. 信息定义
详情视图必须展示：
- **身份信息**:
  - 名称 (如 "警探")
  - 类型 (如 "Investigator / 搜查官")
  - 游戏内图标 (如有)

- **核心属性 (原始值)**:
  - 伤害 (躯干/爆头)
  - 射速 (RPM)
  - 弹匣容量
  - 换弹时间

- **衍生属性 (计算值)**:
  - **DPS (躯干)**: `(RPM / 60) * 躯干伤害`
  - **DPS (爆头)**: `(RPM / 60) * 爆头伤害`
  - **爆发总伤**: `躯干伤害 * 弹匣容量`
  - **TTK (参考)**: 对标准 200HP 目标的击杀时间 (可选)。

## 3. 可视化
- **雷达图**: 嵌入式 300x300px 图表，将该武器的 5 项关键属性与平均值或极值进行对比。

## 4. 交互
- **对比模式**: 提供“对比”开关，允许选择第二把武器叠加在雷达图上，进行直接的 A/B 测试。
