# 首页 / 仪表盘 (Home Dashboard)

**路径**: `/pages/index.vue`
**定位**: 应用程序的着陆页与核心指挥中心。

## 1. 功能概述
首页是用户分析《卡拉彼丘》版本环境 (Meta) 的主要入口，集成了：
- 全局导航
- 数据筛选控制
- 可视化图表 (散点图)
- 详细分析面板 (侧边栏)

## 2. 布局结构
页面采用适合桌面端 (1920x1080) 的弹性布局，并向下兼容移动端。

### 2.1 顶部导航栏 (Header)
- **内容**: 
  - Logo: "STRINOVA ANALYTICS"
  - 元信息: 数据最后抓取时间 (例如 "最后更新: 2023-10-27")
- **交互**: 静态展示。

### 2.2 控制条 (Control Bar)
- **位置**: 悬浮于图表区域的左上角。
- **控件**:
  - **武器类型筛选**: 下拉菜单，用于筛选特定枪种 (如突击步枪、狙击等)。
  - **全局搜索**: 输入框，用于按名称高亮特定武器。

### 2.3 主可视化区 (Left Panel)
- **组件**: `ScatterPlot` (散点图)。
- **行为**: 占据屏幕绝大部分空间，随窗口大小动态缩放。

### 2.4 详情面板 (Right Sidebar)
- **组件**: 自定义侧边栏 (内嵌于 `index.vue`)。
- **行为**:
  - 默认状态: 提示用户选择一把武器。
  - 激活状态: 展示 `RadarChart` (雷达图) 及数值卡片。
