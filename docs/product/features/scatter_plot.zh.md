# 功能模块: 散点图 (Scatter Plot)

**组件路径**: `components/viz/ScatterPlot.vue`

## 1. 目的
可视化“Meta”分布。在射击游戏中，**射速**与**伤害**之间的权衡定义了武器的原型。散点图是揭示“离群值”（统计上更优越的武器，即超模武器）的最佳工具。

## 2. 坐标轴定义
- **X轴**: 射速 (RPM)。线性比例尺。
- **Y轴**: 躯干伤害。线性比例尺。
- **Z轴 (半径)**: 弹匣容量。平方根比例尺 (面积与数值成正比)。
- **颜色**: 武器类型 (分类色盘)。

## 3. 交互设计
- **缩放/平移**: 用户可以使用滚轮缩放至密集区域 (如 600-800 RPM 聚集区)。
- **悬停 (Hover)**: 
  - 目标圆点描边变为白色 (`#FFFFFF`)。
  - 显示 Tooltip，包含名称、DPS、精确的伤害/射速值。
- **点击 (Click)**:
  - 目标圆点描边变为霓虹青 (`#00F0FF`) 并加粗。
  - 触发全局 `selected` 事件。
  - 其他未选中圆点透明度降至 `0.3`。

## 4. 技术约束
- 必须处理重叠点 (解决方案：半透明填充)。
- 必须支持响应式窗口大小调整。
