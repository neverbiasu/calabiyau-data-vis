# 功能模块: 雷达图 (Radar Chart)

**组件路径**: `components/viz/RadarChart.vue`

## 1. 目的
提供武器的多维画像，帮助玩家理解除单纯 DPS 之外的“手感”属性（如：机动性如何？射程多远？）。

## 2. 维度 (坐标轴)
1. **伤害 (Damage)**: 归一化 (基准: 最大伤害 100)。
2. **射速 (Fire Rate)**: 归一化 (基准: 最大射速 1200)。
3. **弹容 (Capacity)**: 归一化 (基准: 最大弹容 100)。
4. **机动 (Mobility)**: 换弹时间的倒数。换弹越快 = 机动分越高。
5. **射程 (Range)**: 归一化有效射程。

## 3. 视觉风格
- **形状**: 多边形 (五边形)。
- **网格**: 5 层同心多边形，分别代表 20%, 40%, 60%, 80%, 100%。
- **填充**: 半透明霓虹青 (`rgba(0, 240, 255, 0.5)`).

## 4. 对比模式
- 当对比两把武器时，第二把武器渲染为“故障红” (`#FF0055`) 的虚线多边形叠加层。
- 重叠区域颜色叠加。
