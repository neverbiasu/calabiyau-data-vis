# 设计规范 (Design System)

## 1. 视觉识别 (Visual Identity)
- **主题**: 赛博朋克 / 战术科幻 (Cyberpunk / Tactical Sci-Fi)。
- **关键词**: 高对比度、霓虹光感、硬朗线条、故障艺术 (Glitch)、数据化。

## 2. 色彩系统 (Color Palette)
以下色值需在 `tailwind.config.js` 中严格配置并使用：

| Token | 色值 (Hex) | 用途说明 |
|-------|-----------|----------|
| `bg-core` | `#08090D` | **全局背景**。深邃的虚空黑，非纯黑，带有极微弱的蓝倾向。 |
| `bg-panel` | `#12141C` | **容器背景**。用于侧边栏、卡片、模态框，略亮于背景以形成层级。 |
| `primary` | `#00F0FF` | **霓虹青**。主色调，用于激活状态、选中高亮、主要数据指标。 |
| `secondary` | `#FF0055` | **故障红**。辅助色，用于对比项、敌对（如有）、高伤害/危险指标。 |
| `accent` | `#FFE600` | **警示黄**。点缀色，用于 Tooltip 关键值、警告信息、特殊标记。 |
| `text-main` | `#EAEAEA` | **主要文本**。灰白色，用于正文，阅读舒适度高。 |
| `text-muted` | `#6B7280` | **次要文本**。深灰色，用于标签、未选中态、坐标轴刻度。 |
| `border-color`| `#2A2F3E` | **边框色**。用于分割线、面板描边。 |

## 3. 排版与字体 (Typography)
- **标题**: `Orbitron` (Google Fonts)
  - 风格: 宽体、科技感强、块状。
  - 用途: Logo、H1、H2、模块大标题。
- **正文**: `Rajdhani` (Google Fonts)
  - 风格: 直角的无衬线体，紧凑，具有工业感。
  - 用途: 正文、数据标签、按钮文字。
- **数字**: 推荐使用等宽字体 (Monospace) 或 `Rajdhani` 的等宽特性，确保表格中数值对齐。

## 4. 组件规范 (Components)

### 4.1 按钮 (Buttons)
- **外形**: 45度切角设计 (使用 `clip-path` 或伪元素实现)，1px 边框。
- **状态**:
  - **Default**: 透明背景 + 边框色描边 + 文字主色。
  - **Hover**: 增加外发光 `box-shadow: 0 0 8px rgba(0, 240, 255, 0.4)`，背景微亮。
  - **Active**: 填充主色，文字变黑。

### 4.2 图表 (Charts)
- **散点图 (Scatter)**:
  - **点**: 默认为空心圆环，Hover 时变为实心并放大。
  - **网格**: 虚线，极低透明度 (`opacity: 0.1`)。
  - **坐标轴**: 隐藏轴线，仅保留刻度值。
- **雷达图 (Radar)**:
  - **区域**: 填充透明度 `0.4`，描边宽度 `2px`。
  - **网格**: 多边形同心圆，而非圆形。

## 5. 交互动效 (Micro-Interactions)
- **Tooltip**: 跟随鼠标移动，启用 `backdrop-blur` (毛玻璃) 效果，边框发光。
- **Transition**: 所有颜色变化、大小缩放需添加 `duration-200 ease-out` 过渡，避免生硬跳变。
